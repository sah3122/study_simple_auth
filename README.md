# study_simple_auth
Spring Security를 활용한 Auth 서비스

---

* Spring Boot 2.2.2
* Spring Security 5.2.1
* Spring Data JPA 2.2.3
---

* Username Password 기반 인증 구현
* Redis Session Registry 적용
  * Docker Redis 사용
* OAuth2 Server 구현
    * Inmemory Token Store 사용
    * Inmemory client 사용
